# Campus Navigation System

## 项目简介

校园导览系统是一个基于 C++ 实现的数据结构课程设计项目。

本项目将校园地图抽象为图结构：

- 校园地点 → 图的顶点(Vertex)
- 校园道路 → 图的边(Edge)
- 道路距离 → 边的权值(Weight)

通过图论相关算法，实现校园地点查询、道路查询以及最短路径导航功能。

---

## 项目目标

实现一个简单的校园地图导航系统，使用户能够：

1. 查询校园地点信息
2. 查询地点之间的道路关系
3. 查询任意两个地点之间的最短路径
4. 输出推荐导航路线

后续扩展：

- 校园地图可视化
- 最短路线高亮显示
- 用户查询记录保存
